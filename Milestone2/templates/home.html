Automated E2E Testing

This system uses a natural language parser to convert human-readable test instructions into executable automation steps. The parser identifies actions such as navigation, clicks, form inputs, and assertions, then maps them to browser-based test commands.

This allows testers to write test cases in plain English while the automation engine handles execution and validation behind the scenes.
