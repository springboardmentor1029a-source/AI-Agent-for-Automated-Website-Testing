About This Project

   

A modern AI-powered automation tool that turns natural language into browser tests.

 
 
   

What This System Does

   

This AI-based system allows anyone to test a website without writing code. Just describe the action in plain English and the agent will convert it into automated steps, run them using Playwright, and provide a detailed report including pass/fail metrics and failure analysis.

 
 
   

Technology Stack

   
         
  • Python + Flask (backend)
    •      
  • **LangGraph** (AI Workflow Orchestration)
    •      
  • **Playwright** (Browser Automation)
    •      
  • **SQLite** (Test History Database)
    •      
  • NLP for instruction parsing (Simulated/Planned LLM)
    •    
 
 
   

Who Can Benefit

   

Developers, QA engineers, students learning automation, and product teams who want fast validation without investing in heavy test code. The LLM-driven flow ensures high adaptivity and robust reporting.
