About This Project

A modern AI-powered automation tool that turns natural language into browser tests.

What This System Does

This AI-based system allows anyone to test a website without writing code. Just describe the action in plain English and the agent will convert it into automated steps and run them.

Technology Stack

  • Python + Flask (backend)
  • Playwright (browser automation)
  • HTML/CSS/JS frontend
  • NLP for instruction parsing (planned)

Who Can Benefit

Developers, QA engineers, students learning automation, and product teams who want fast validation without investing in heavy test code.
