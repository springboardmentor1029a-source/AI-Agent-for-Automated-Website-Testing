Documentation

Getting Started

Follow these steps to start using the Web Test Agent:

  1. Navigate to the Test Console page
  2. Enter your test instruction in natural language
  3. Click "Send" or press Enter
  4. Review the agent's response

API Endpoints

POST /api/agent

Send instructions to the LangGraph agent

{
  "input": "Your test instruction here"
}

Response:

{
  "status": "success",
  "message": "Agent response",
  "input": "Original input"
}

Configuration

The platform uses environment variables for configuration:

  • OPENAI_API_KEY - Your OpenAI API key
  • PORT - Server port (default: 5000)
  • DEBUG - Enable debug mode (True/False)

Test Elements Available

The test console provides the following interactive elements:

  • Buttons (Primary, Secondary, Success, Danger)
  • Form inputs (Text, Email, Textarea)
  • Navigation links
  • AI Agent interface

Requirements

System requirements and dependencies:

  • Python 3.8 or higher
  • Flask web framework
  • LangGraph library
  • OpenAI API access
  • python-dotenv for environment management
